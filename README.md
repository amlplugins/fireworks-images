# @amlplugins/fireworks-images

Native `@ai-sdk/fireworks` SDK re-exported under the `@amlplugins` namespace.

> Fireworks Images — FLUX.1 [schnell/dev], SDXL, Playground v2.5 via @ai-sdk/fireworks imageModel adapter.

## Install

```bash
npm install @amlplugins/fireworks-images
```

## Use

```ts
import { /* named exports from @ai-sdk/fireworks */ } from "@amlplugins/fireworks-images";
// or
import sdk from "@amlplugins/fireworks-images";
```

This package additionally re-exports `ai` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
